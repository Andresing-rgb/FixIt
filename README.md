# App de Motivos de Rechazo
Extensión de navegador para gestionar plantillas de **motivos de rechazo** en trámites de Apostilla y Legalización del Ministerio de Relaciones Exteriores.

Versión actual: **1.0.0**

---

# 🎯 Objetivo

Facilitar a los funcionarios la gestión y utilización de plantillas de respuesta para solicitudes de Apostilla y Legalización, permitiendo:

- Buscar plantillas rápidamente
- Copiar contenido
- Insertar texto directamente en formularios web
- Administrar plantillas
- Importar plantillas masivamente desde archivos JSON

---

# 🧠 Funcionalidades principales

✔ Gestión de plantillas  
✔ Búsqueda rápida por:

- código
- motivo de rechazo
- título
- texto

✔ Copiar plantilla  
✔ Insertar plantilla en formulario activo  
✔ Activar / desactivar plantillas  
✔ Importación masiva mediante JSON  
✔ Almacenamiento local mediante `chrome.storage.local`  
✔ Interacción con páginas web mediante `content.js`
